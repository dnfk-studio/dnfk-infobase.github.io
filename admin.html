Admin：上傳 PDF 檔案

此頁面需要先在 assets/js/config.js 設定 Google Apps Script Web App URL。建議用 Cloudflare Access 保護此頁。

上傳說明

  1. 在 Google 雲端硬碟建立資料夾，取得其 Folder ID。
  2. 建立 Google 試算表，第一列欄位為：id, title, date, tags, desc, url。
  3. 佈署 Apps Script（見 apps_script/README_APPS_SCRIPT.md）。
  4. 將 Web App URL 貼到 assets/js/config.js 的 APPS_SCRIPT_URL。
